# Guide

## Getting Started

- [What is Agent Mesh?](../docs/documentation/getting-started/introduction.md): An introduction to Agent Mesh, an open-source framework that serves as an integration layer for AI applications. It explains how Agent Mesh uses event-driven architecture to enable specialized agents to collaborate across isolated data silos, featuring unified AI collaboration, complex workflow orchestration, seamless system integration, and enterprise-grade reliability built on Solace Event Broker.

- [Getting Started](../docs/documentation/getting-started/getting-started.md): Overview of Agent Mesh and guide to navigating the documentation, providing developers with a roadmap for understanding the framework's capabilities and structure.

- [Architecture](../docs/documentation/getting-started/architecture.md): Detailed explanation of Agent Mesh's architectural design, including its event-driven core, component interactions, and how the event mesh creates a neural network for AI components to communicate asynchronously.

- [Try Agent Mesh](../docs/documentation/getting-started/try-agent-mesh.md): Quick start guide for creating your first project, building, and running Agent Mesh to get hands-on experience with the framework.
